# CodeUp Mobile IDE
A streamlined, touch-first mobile IDE designed for on-the-go development.

## Core Architecture
* **Editor Logic:** Handles text-to-syntax conversion and line tracking.
* **UI Engine:** Manages modals, the floating notepad, and layout state.
* **API Handler:** Manages asynchronous communication with build servers.

## Getting Started
1. Clone the repository.
2. Initialize your local terminal server (Node.js).
3. Open `index.html` in your mobile browser.
